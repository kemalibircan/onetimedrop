## Why Public Computers Are Risky

A public computer is never just a screen and a keyboard. It may keep browser downloads, recent documents, print history, autofill data, and temporary files after you walk away. The next user may not see everything, but you should assume the machine remembers more than it shows.

That does not mean you can never use one. It means you need a shorter, cleaner workflow.

## What Can Be Left Behind

The most common leftovers are:

- Files in the Downloads folder
- Open tabs with active cloud accounts
- PDF previews and print queues
- Browser autofill, cookies, and cached pages
- USB drive metadata or copied file paths

If the machine is poorly maintained, even deleted files may be recoverable later.

## Safer Workflow for Shared Computers

1. Transfer only the exact files you need.
2. Prefer temporary browser tools like [OneTimeDrop](/) instead of signing into your full cloud drive.
3. Use a private or incognito window whenever possible.
4. Download, print, or upload the file, then remove it immediately.
5. Empty the browser session and confirm the file is no longer in Downloads before leaving.

## Better Than Logging Into Everything

Logging into email, Google Drive, or your full photo account on a public machine creates a wider security surface. A temporary session-based transfer limits exposure because you are not opening years of stored data just to get one document.

| Method | Exposure level | Why |
|---|---|---|
| Full cloud login | High | Access to many unrelated files |
| Email attachment | Medium | Leaves inbox session and downloads |
| Session transfer | Lower | Narrow, short-lived access |
| USB drive | Medium | Can still leave copies on the computer |

> Practical rule: if the file is legal, medical, financial, or personal, treat cleanup as part of the task, not an optional final step.

## FAQ

**Q: Is printing from a copy shop always unsafe?**  
A: Not always, but it is never zero-risk. Use the shortest possible workflow and clear the file after printing.

**Q: Can browser history reveal what I opened?**  
A: Yes. Recent history, cached previews, and downloaded filenames may all reveal activity.

**Q: Is a QR-based transfer safer than email on a public computer?**  
A: Usually yes for one-off transfers, because you do not need to log into a permanent inbox or cloud account.
