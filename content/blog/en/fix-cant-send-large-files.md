## Why Can't I Send Large Files?

Every sharing platform has limits. Email typically caps attachments at 25MB. WhatsApp compresses photos and limits videos. iMessage struggles with files over 100MB. And Bluetooth transfers large files painfully slowly.

Understanding the limits helps you pick the right tool.

## Common Large File Problems & Fixes

### Problem 1: "File too large" error in email

**Cause:** Gmail/Outlook/etc. limit attachments to 25MB.

**Fix options:**
- Use a cloud storage link (Google Drive, Dropbox, iCloud) — paste the share link instead of attaching the file
- For one-time transfers under 50MB, use [OneTimeDrop](/) — no email account needed
- Compress the file first: zip it on your phone to reduce size

### Problem 2: WhatsApp compresses my images

**Cause:** WhatsApp deliberately reduces image quality to save bandwidth.

**Fix:** Don't use WhatsApp for quality-critical photos. Use OneTimeDrop or AirDrop (between Apple devices) to preserve original quality.

### Problem 3: Video file is too large to share

**Video tips:**
- Compress using a free tool like HandBrake (desktop) or Compress Video (iOS/Android)
- Trim to just the relevant portion
- For under 50MB, OneTimeDrop works. For larger files, WeTransfer (up to 2GB free) is best.

### Problem 4: USB drive not available

If you don't have a USB cable or drive:
- Use browser transfer: [OneTimeDrop](/) works without any cables
- Make sure both devices have internet (mobile data works)
- Transfer up to 20 files, 50MB each

### Problem 5: Phone and computer won't pair over Bluetooth

**Common causes:** Bluetooth pairing varies by OS; Windows and Android don't always pair smoothly.

**Fix:** Skip Bluetooth entirely. Use a Wi-Fi based browser transfer like OneTimeDrop. Bluetooth is designed for audio/peripherals, not file transfer.

## Choosing the Right Tool By File Size

| File size | Best tool |
|---|---|
| Under 25MB | Email, OneTimeDrop, WhatsApp |
| 25MB–50MB | OneTimeDrop, cloud link |
| 50MB–2GB | WeTransfer, cloud storage link |
| Over 2GB | USB drive, Google Drive paid |
