## Wi-Fi File Transfer: Why "No Install" Matters

When you need to transfer files on someone else's computer, you usually can't install software. That makes traditional Wi-Fi transfer apps useless in those situations.

Browser-based tools fill this gap perfectly — they run in the existing web browser with no installation required.

## How Browser-Based Wi-Fi Transfer Works

Tools like [OneTimeDrop](/) use your internet connection (Wi-Fi or mobile data) as the transfer medium. Here's the flow:

1. Computer opens a session → generates a code
2. Phone scans the QR or enters the code → joins the session
3. Files upload from phone → server → available to download on computer

The key insight: **the two devices don't need to be on the same Wi-Fi network**. Each just needs its own internet connection.

## Step-by-Step Guide

### On Any Computer (No Install Needed)

1. Open any browser (Chrome, Firefox, Safari, Edge)
2. Go to [onetimedrop.io](/)
3. A QR code and 8-digit code appear immediately
4. Keep this tab open

### On Your Phone (No App Needed)

5. Open your camera (iPhone) or QR scanner (Android)
6. Scan the QR code
7. The join page opens automatically in your phone's browser
8. Tap **Select files** — pick what you want to transfer
9. Upload — watch the progress bar

### Download (Computer)

10. Files appear in the list with a Download button
11. Click to save each file — they arrive at original quality

## Privacy & Safety Tip

> ⚠️ **Disclaimer:** OneTimeDrop transmits files over the internet (not peer-to-peer). Files are accessible only with your session token and are auto-deleted after 10 minutes. Don't transfer sensitive documents on public computers.

## FAQ

**Q: Does this work if the devices are on different Wi-Fi networks?**  
A: Yes — they just each need any internet access.

**Q: Does this work on mobile data?**  
A: Yes. Mobile data works for both devices.

**Q: Is there a speed limit?**  
A: Transfer speed depends on your internet connection. A 5MB file typically transfers in 1–5 seconds.
