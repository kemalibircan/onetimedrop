## El formato importa para la impresión

Has enviado el archivo. Estás frente a la computadora. Presionas Imprimir. Y el resultado parece incorrecto: el texto está borroso, el diseño está cambiado y las imágenes están comprimidas. ¿El problema? Formato de archivo incorrecto.

Esto es qué usar y cuándo.

## PDF: Casi siempre la mejor opción

**PDF (Formato de documento portátil)** se inventó literalmente para imprimir. Bloquea sus fuentes, diseño, imágenes y márgenes. Lo que ves es exactamente lo que se imprime.

✅ Uso para:
- Cualquier documento con texto (cartas, CV, informes, formularios)
- Presentaciones (exportadas desde PowerPoint/Keynote)
- Documentos escaneados
- Billetes y tarjetas de embarque.
- Facturas

**Cómo crear PDF en iPhone:**
Cualquier aplicación (Word, Notas, Pages) → Compartir → Imprimir → Pellizcar para expandir la vista previa → Compartir nuevamente → Guardar como PDF

**Cómo crear PDF en Android:**
Google Docs/Office → Compartir → Imprimir → Guardar como PDF (seleccionar de la lista de impresoras)

## JPEG: bueno para fotos, no para texto

**JPEG (.jpg)** es excelente para fotografías, pero terrible para cualquier cosa con texto o líneas nítidas, porque:
- JPEG utiliza compresión con pérdida: las imágenes se vuelven ligeramente borrosas
- El texto y las líneas nítidas se ven "turbios" en la configuración de calidad JPEG
- Cada nuevo guardado degrada aún más la calidad

✅ Uso para:
- Impresión de fotografías.
- Libros de fotos
- Impresiones de pared

❌ Evitar para:
- Documentos con texto
- Logotipos o diagramas
- Presentaciones

## PNG: excelente para la transparencia, excesivo para las fotos

**PNG** utiliza compresión sin pérdidas: la calidad se conserva perfectamente. Es genial para capturas de pantalla, logotipos y cualquier cosa con capas de transparencia. Pero los archivos PNG son más grandes que los JPEG, lo que puede ralentizar ligeramente la transferencia.

✅ Uso para:
- Capturas de pantalla
- Gráficos con fondos transparentes.
- Logotipos, iconos

## HEIC: formato de iPhone: necesita conversión

**HEIC** es el formato fotográfico de Apple que ahorra espacio. Es excelente para el almacenamiento, pero no es universalmente compatible:
- Es posible que Windows no abra HEIC sin la extensión del códec HEIC
- Las impresoras y el software más antiguos no lo admiten.

**Solución:** Antes de imprimir, convierta HEIC a JPEG:
- iPhone: Configuración → Cámara → Formatos → Más compatible (fuerza la captura JPEG)
- O utilice una aplicación de conversión gratuita

## DOCX/PPTX: Sólo si la computadora tiene Microsoft Office

Los archivos de Word y PowerPoint se imprimirán bien SI la computadora tiene Office instalado. Si no:
- El archivo puede abrirse en LibreOffice con problemas de diseño.
- Algunas impresoras no verán estos formatos en absoluto

**Elección más segura:** Exportar a PDF antes de enviar.

## Preguntas frecuentes

**P: Tengo un PNG pero necesito imprimir un póster gigante. ¿Se verá bien?**  
R: PNG no tiene pérdidas, por lo que la calidad no se degradará. Sin embargo, asegúrese de que la resolución de la imagen (DPI) sea lo suficientemente alta para el tamaño de impresión.

**P: ¿Puedo convertir un documento de Word a PDF en mi teléfono antes de transferirlo?**  
R: Sí. Microsoft Word Mobile y Google Docs ofrecen exportación de PDF.

**P: ¿OneTimeDrop convierte tipos de archivos?**  
R: No. OneTimeDrop transfiere archivos en su formato original sin modificaciones.
