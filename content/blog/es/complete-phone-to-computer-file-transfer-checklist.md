## Por qué ayuda una lista de verificación

La mayoría de las fallas en la transferencia de archivos son predecibles: el archivo tiene el formato incorrecto, el método se elige demasiado tarde, la computadora se comparte o el remitente usa una herramienta que comprime los medios. Una lista de verificación previene esos errores antes de que comience el tiempo.

Úselo antes de cualquier transferencia importante de teléfono a computadora.

## La lista de verificación

### 1. Identificar el archivo real
Asegúrese de tener la versión final, no una captura de pantalla, un borrador o una copia reenviada.

### 2. Elija el formato correcto
Para documentos, prefiera PDF. Para editar, conserve el archivo de foto o vídeo original.

### 3. Haga coincidir el método con el momento
Utilice [OneTimeDrop](/) para transferencias cercanas, enlaces vencidos para entrega remota y cable o USB para un verdadero respaldo sin conexión.

### 4. Piensa en el dispositivo
If the receiving computer is borrowed or public, avoid logging into broad personal accounts.

### 5. Confirma la calidad después de la descarga.
Verifique el tamaño del archivo, abra el archivo y verifique las páginas o la reproducción multimedia.

### 6. Limpiar
Elimine las descargas sobrantes en máquinas compartidas y cierre la sesión de transferencia.

## Tabla de referencia rápida

| Situación | Mejor primera opción |
|---|---|
| Computadora portátil cercana | Transferencia de navegador basada en sesión |
| Destinatario remoto | Enlace caducado |
| Gran lote fuera de línea | Cable o USB |
| Tarea sensible de computadora compartida | Sesión temporal más limpieza |

## Preguntas frecuentes

**P: ¿Cuál es el error más común?**  
R: Usar el método más familiar en lugar del más apropiado.

**P: ¿Debo evitar siempre el correo electrónico?**  
R: No, pero evítelo para archivos grandes, confidenciales o urgentes cuando existan mejores opciones.

**P: ¿Esta lista de verificación es solo para el trabajo?**  
R: No. También ayuda con los viajes, la escuela, la impresión, las reuniones y los documentos familiares.
