## Qu'est-ce qui rend un ordinateur « inconnu » ?

Un ordinateur inconnu est tout ordinateur qui n'est pas le vôtre et qui n'est pas entièrement sous votre contrôle :
- Centre d'affaires de l'hôtel
- Poste de travail bibliothèque
- Terminal de photocopie
- Le PC d'un ami ou d'un membre de la famille
- Ordinateur de bureau d'un autre service

Les risques liés à l’utilisation de ces ordinateurs concernent à la fois ce que vous laissez derrière vous (fichiers, historique du navigateur) et ce que l’ordinateur peut exécuter dessus (logiciels malveillants, enregistreurs de frappe).

## Étape 1 : Évaluer le niveau de risque

**Faible risque :** Ordinateur portable personnel d'un ami. Ils sont à proximité, vous leur faites confiance, c'est temporaire.

**Risque moyen :** Ordinateur de bureau géré par le service informatique. Propre, mais le service informatique de l'entreprise peut disposer d'un logiciel de surveillance.

**Risque élevé :** Ordinateur de bibliothèque ou d'hôtel. Utilisateurs multiples, logiciels inconnus, enregistreurs de frappe potentiels. Minimisez ce que vous faites ici.

## Étape 2 : Utiliser le mode Privé/Incognito

Avant d'ouvrir un onglet du navigateur pour transférer des fichiers, passez en mode Incognito/Privé :
- **Chrome :** Ctrl+Maj+N (Cmd+Maj+N sur Mac)
- **Firefox :** Ctrl+Maj+P
- **Safari :** Cmd+Maj+N

Cela empêche l'ordinateur d'enregistrer l'historique du navigateur, les cookies, les mots de passe ou les données de saisie automatique de votre session.

## Étape 3 : Transférez votre fichier

Utilisez [OneTimeDrop](/) — cela ne nécessite pas de vous connecter à un compte :

1. Ouvrez OneTimeDrop en mode navigation privée
2. Le code de session et le QR apparaissent
3. Sur votre téléphone, rejoignez la session et téléchargez le(s) fichier(s)
4. Téléchargez sur l'ordinateur

Aucun mot de passe saisi. Aucune session de compte laissée pour compte.

## Étape 4 : utilisez le fichier, puis supprimez-le

Après avoir téléchargé et utilisé votre fichier (impression, ouverture, etc.) :

1. **Supprimer du dossier Téléchargements** — Maj+Supprimer (Windows) ou clic droit → Déplacer vers la corbeille (Mac)
2. Si vous avez vidé la corbeille, le fichier disparaît du système de fichiers (bien qu'il ne soit pas irrécupérable d'un point de vue médico-légal, mais convient pour une utilisation typique)
3. Fermez tous les onglets du navigateur

## Étape 5 : Déconnectez-vous et effacez toutes les sessions

- Fermez l'onglet OneTimeDrop - votre session expire automatiquement dans 10 minutes de toute façon
- Si vous vous connectez accidentellement à un site Web, déconnectez-vous et révoquez les sessions de votre téléphone

## Conseil de confidentialité et de sécurité

> ⚠️ **Avertissement :** OneTimeDrop supprime automatiquement les fichiers du serveur après 10 minutes. Cependant, les fichiers que vous téléchargez restent sur le disque dur de l'ordinateur jusqu'à ce qu'ils soient supprimés manuellement. Sur les ordinateurs publics, supprimez toujours le fichier immédiatement après utilisation.

## Et si le fichier est sensible ?

Si le dossier contient des informations sensibles (document juridique, dossier médical) :
- Choisissez uniquement des ordinateurs à risque moyen ou de meilleure qualité
- Utilisez les données mobiles, pas le Wi-Fi inconnu
- Supprimer après utilisation, puis vider la Corbeille/Corbeille

##FAQ

**Q : Le service informatique peut-il voir les fichiers que je transfère sur son ordinateur ?**  
R : Potentiellement oui : le service informatique peut surveiller le trafic réseau et les agents installés peuvent enregistrer l'activité des fichiers. Utilisez votre propre appareil pour les transferts sensibles.

**Q : Le mode navigation privée me protège-t-il des enregistreurs de frappe ?**  
R : Non. La navigation privée empêche uniquement le stockage de l'historique/de la session du navigateur. Les keyloggers fonctionnent au niveau du système d’exploitation.

**Q : Quel est le moyen le plus sûr de déplacer des fichiers sur un ordinateur inconnu ?**  
R : Utilisez un réseau de confiance, la navigation privée, transférez ce dont vous avez besoin, supprimez immédiatement après.
