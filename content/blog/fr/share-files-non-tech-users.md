## Le défi : aider les utilisateurs non techniques à recevoir des fichiers

Les utilisateurs techniques sous-estiment souvent à quel point le partage de fichiers peut être déroutant pour les autres. « Envoyez-le simplement sur WhatsApp » – mais que se passe-t-il s'ils ne trouvent pas le fichier ? "Télécharger sur Google Drive" — mais ils n'ont pas de compte.

La méthode idéale pour les utilisateurs non techniques est :
- Implique le moins d'étapes possible
- Ne nécessite pas de création de compte
- A une frappe minimale
- Leur donne un signal de réussite clair

## Les meilleures méthodes pour les utilisateurs non-technologiques

### Méthode 1 : Code QR (zéro saisie requise)

La méthode la plus simple pour partager en personne (même pièce) :

1. Vous ouvrez [onetimedrop.io](/) sur leur ordinateur
2. Un code QR apparaît à l'écran
3. Ils le scannent avec l'appareil photo de leur téléphone : la page de participation s'ouvre automatiquement
4. Vous leur dites : "Sélectionnez le fichier dans votre application Fichiers ou Photos et appuyez sur Télécharger"
5. Ils le téléchargent sur l’ordinateur

La seule technologie qu’ils doivent connaître : comment ouvrir leur application appareil photo.

### Méthode 2 : envoyer un e-mail avec un lien de téléchargement direct

Pour les utilisateurs distants non techniques, le courrier électronique reste fiable :
- Envoyer le fichier en pièce jointe (moins de 25 Mo)
- Ils cliquent sur la pièce jointe dans leur boîte de réception : elle se télécharge automatiquement

Simple, familier à la plupart des gens, aucune nouvelle application à apprendre. L'inconvénient : la qualité, les limites de taille et la permanence de la boîte de réception.

### Méthode 3 : WhatsApp (s'ils y sont déjà)

Si la personne utilise WhatsApp régulièrement, y envoyer une photo ou un fichier ne nécessite aucun nouvel apprentissage de sa part. Le compromis est la compression des fichiers et la confidentialité, mais pour un partage quotidien à faibles enjeux, cela fonctionne.

### Méthode 4 : Racontez chaque étape

Lorsque vous aidez un utilisateur non technique avec OneTimeDrop :

1. "Ouvrez le site onetimedrop.io sur l'ordinateur" (ou faites-le pour eux)
2. "Maintenant sur votre téléphone, ouvrez votre appareil photo et pointez-le vers ce carré de l'écran"
3. "Appuyez sur la notification qui apparaît"
4. « Voyez-vous un bouton indiquant « Sélectionner les fichiers » ? Appuyez dessus »
5. "Choisissez la photo que vous souhaitez envoyer"
6. "Maintenant, regardez l'écran de l'ordinateur : votre photo devrait y apparaître"

Des conseils verbaux étape par étape éliminent toute confusion.

## Ce qui ne fonctionne pas avec les utilisateurs non-technologiques

- Tout ce qui implique la création de comptes
- Méthodes nécessitant des pilotes ou des installations d'applications
- Services avec trop d'options/paramètres
- Appairage Bluetooth (messages d'erreur déroutants)

## Quel format est le plus facile à ouvrir pour les utilisateurs non-technologiques ?

Pour les photos : **JPEG** – universellement compris, s'ouvre dans n'importe quelle visionneuse de photos.  
Pour les documents : **PDF** — « cliquez simplement dessus, il s'ouvre ».  
Évitez HEIC, DOCX, PPTX si la personne n'a pas Office ou Mac.

> ⚠️ **Avertissement :** OneTimeDrop supprime automatiquement les fichiers après 10 minutes. Guidez l'utilisateur non technique dans le téléchargement du fichier avant l'expiration de la session.

##FAQ

**Q : Mon parent âgé ne sait pas comment utiliser un scanner QR avec appareil photo. Que dois-je faire ?**  
R : Tapez simplement le code à 8 chiffres manuellement sur [onetimedrop.io/join](/join). Lisez-le-leur simplement par téléphone.

**Q : Que se passe-t-il s'ils ferment accidentellement le navigateur ?**  
R : Tant que la session est toujours active, ils peuvent rouvrir OneTimeDrop sur l'ordinateur et les fichiers seront toujours là.

**Q : Existe-t-il une version de OneTimeDrop qui fonctionne sur les anciens navigateurs ?**  
R : OneTimeDrop nécessite un navigateur moderne (Chrome, Firefox, Edge, Safari à partir de 2020). Les très anciens navigateurs IE ne fonctionneront pas.
