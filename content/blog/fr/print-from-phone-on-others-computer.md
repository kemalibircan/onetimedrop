## Le problème du Copy Shop

Vous arrivez dans une imprimerie avec un document enregistré sur votre téléphone. L'employé montre un ordinateur. Votre téléphone ne partage pas de réseau avec lui. Vous n'avez pas de clé USB. Vous ne souhaitez pas vous connecter à votre messagerie sur une machine publique.

Cela vous semble familier ? Il s’agit de l’une des frustrations les plus courantes dans la vie numérique quotidienne, et la solution est plus simple que vous ne le pensez.

## Le moyen le plus simple : code de session du navigateur

Des outils comme [OneTimeDrop](/) placent un code de session et un code QR sur l'écran de l'ordinateur à partir duquel vous souhaitez imprimer. Vous le numérisez avec votre téléphone, téléchargez votre document et le téléchargez sur l'ordinateur, prêt à être imprimé.

Aucun compte. Pas de câbles USB. Pas d'envoi d'e-mail.

## Étape par étape dans une imprimerie ou une bibliothèque

### Sur l'ordinateur (copieuse)

1. Ouvrez un navigateur et accédez à [onetimedrop.io](/)
2. Un grand code QR et un code à 8 chiffres apparaissent à l'écran
3. Ne fermez pas cet onglet – gardez-le visible

### Sur votre téléphone

4. Ouvrez votre application Appareil photo (iPhone/Android moderne) et scannez le code QR.
5. Ou allez sur [onetimedrop.io/join](/join) et tapez le numéro à 8 chiffres
6. Appuyez sur **Sélectionner les fichiers** et choisissez votre document (PDF, Word, etc.)
7. Attendez la fin du téléchargement (secondes)

### De retour sur l'ordinateur

8. Le fichier apparaît dans la liste des fichiers
9. Cliquez sur **Télécharger** : le fichier est enregistré sur le bureau.
10. Ouvrez-le et imprimez normalement

## Quels types de fichiers puis-je envoyer pour impression ?

- ✅ PDF — le meilleur choix pour l'impression
- ✅DOCX (Mot)
- ✅PPTX (PowerPoint)
- ✅ Images : JPEG, PNG, HEIC

Préférez toujours **PDF** pour l'impression : il préserve le formatage quel que soit le logiciel qui l'ouvre.

## Confidentialité sur les ordinateurs publics

> ⚠️ **Avertissement :** Les fichiers OneTimeDrop sont automatiquement supprimés après 10 minutes. Soyez toutefois prudent sur les ordinateurs publics. Les fichiers que vous téléchargez peuvent rester dans le dossier Téléchargements jusqu'à ce qu'ils soient supprimés manuellement. Supprimez toujours les fichiers téléchargés des ordinateurs partagés après l'impression.

### Liste de contrôle de sécurité pour les ordinateurs publics
- Utilisez le mode Privé/Incognito lorsque cela est possible
- Supprimez votre fichier téléchargé des téléchargements après l'impression
- Ne saisissez pas de mots de passe ou de données personnelles
- Fermez tous les onglets du navigateur lorsque vous avez terminé

## Questions fréquemment posées

**Q : Que se passe-t-il si l'ordinateur du photocopieur ne parvient pas à ouvrir les fichiers PDF ?**  
R : La plupart des ordinateurs modernes intègrent la prise en charge des fichiers PDF. Vous pouvez également imprimer directement à partir de la visionneuse intégrée de Chrome.

**Q : Que faire si l'ordinateur n'a pas Internet ?**  
R : OneTimeDrop a besoin d’Internet sur les deux appareils. Si l'ordinateur n'a pas Internet, demandez au magasin si une option USB est disponible.

**Q : Comment puis-je convertir mon fichier en PDF sur mon téléphone ?**  
R : Sur iPhone : utilisez le raccourci « Imprimer au format PDF » dans n'importe quelle application. Sur Android : la plupart des applications Office vous permettent d'exporter directement au format PDF.

**Q : Le code à 8 chiffres est-il sûr ?**  
R : Le code est généré aléatoirement et expire dans 10 minutes. Seule la personne possédant le code peut rejoindre votre session.
