## Le format est important pour l'impression

Vous avez envoyé le fichier. Vous êtes devant l'ordinateur. Vous appuyez sur Imprimer. Et le résultat semble faux : le texte est flou, la mise en page est décalée, les images sont compressées. Le problème ? Mauvais format de fichier.

Voici quoi utiliser et quand.

## PDF : presque toujours le meilleur choix

**Le PDF (Portable Document Format)** a été littéralement inventé pour l'impression. Il verrouille vos polices, votre mise en page, vos images et vos marges. Ce que vous voyez est exactement ce qui s'imprime.

✅ Utiliser pour :
- Tout document comportant du texte (lettres, CV, rapports, formulaires)
- Présentations (exportées depuis PowerPoint/Keynote)
- Documents numérisés
- Billets et cartes d'embarquement
- Factures

**Comment créer un PDF sur iPhone :**
N'importe quelle application (Word, Notes, Pages) → Partager → Imprimer → Pincer pour agrandir l'aperçu → Partager à nouveau → Enregistrer au format PDF

**Comment créer un PDF sur Android :**
Google Docs/Office → Partager → Imprimer → Enregistrer au format PDF (sélectionner dans la liste des imprimantes)

## JPEG : bon pour les photos, pas pour le texte

**JPEG (.jpg)** est excellent pour les photographies mais terrible pour tout ce qui contient du texte ou des dessins au trait nets, car :
- JPEG utilise une compression avec perte : les images deviennent légèrement floues
- Le texte et les lignes nettes semblent « boueux » avec les paramètres de qualité JPEG
- Chaque nouvelle sauvegarde dégrade davantage la qualité

✅ Utiliser pour :
- Impression de photographies
- Livres photos
- Impressions murales

❌ À éviter pour :
- Documents avec texte
- Logos ou schémas
- Présentations

## PNG : idéal pour la transparence, excessif pour les photos

**PNG** utilise une compression sans perte : la qualité est parfaitement préservée. C'est idéal pour les captures d'écran, les logos et tout ce qui comporte des couches de transparence. Mais les fichiers PNG sont plus volumineux que les fichiers JPEG, ce qui peut légèrement ralentir le transfert.

✅ Utiliser pour :
- Captures d'écran
- Graphiques avec arrière-plans transparents
- Logos, icônes

## HEIC : format de l'iPhone – nécessite une conversion

**HEIC** est le format photo peu encombrant d'Apple. C'est idéal pour le stockage, mais pas universellement pris en charge :
- Windows ne peut pas ouvrir HEIC sans l'extension du codec HEIC
- Les anciennes imprimantes et logiciels ne le prennent pas en charge

**Correction :** Avant d'imprimer, convertissez HEIC en JPEG :
- iPhone : Paramètres → Appareil photo → Formats → Les plus compatibles (force la capture JPEG)
- Ou utilisez une application de conversion gratuite

## DOCX / PPTX : uniquement si l'ordinateur dispose de Microsoft Office

Les fichiers Word et PowerPoint s'imprimeront correctement SI Office est installé sur l'ordinateur. Sinon :
- Le fichier peut s'ouvrir dans LibreOffice avec des problèmes de mise en page
- Certaines imprimantes ne verront pas du tout ces formats

**Choix plus sûr :** Exportez au format PDF avant l'envoi.

## Transférez votre fichier

Une fois que vous avez le bon format, utilisez [OneTimeDrop](/) pour transférer instantanément du téléphone à l'ordinateur, puis imprimez.

> ⚠️ **Avertissement :** OneTimeDrop transfère les fichiers tels quels. Vérifiez toujours l'aperçu avant impression sur l'ordinateur avant d'imprimer, en particulier pour les documents de plusieurs pages.

##FAQ

**Q : J'ai un fichier PNG mais j'ai besoin d'imprimer une affiche géante : est-ce que ça aura l'air correct ?**  
R : Le PNG est sans perte, donc la qualité ne se dégradera pas. Cependant, assurez-vous que la résolution de l'image (DPI) est suffisamment élevée pour la taille d'impression.

**Q : Puis-je convertir un document Word en PDF sur mon téléphone avant de le transférer ?**  
R : Oui. Microsoft Word Mobile et Google Docs proposent tous deux l'exportation au format PDF.

**Q : OneTimeDrop convertit-il les types de fichiers ?**  
R : Non. OneTimeDrop transfère les fichiers dans leur format d'origine sans modification.
