## Transfert de fichiers Wi-Fi : pourquoi « aucune installation » est importante

Lorsque vous devez transférer des fichiers sur l'ordinateur de quelqu'un d'autre, vous ne pouvez généralement pas installer de logiciel. Cela rend les applications de transfert Wi-Fi traditionnelles inutiles dans ces situations.

Les outils basés sur un navigateur comblent parfaitement cette lacune : ils s'exécutent dans le navigateur Web existant sans aucune installation requise.

## Comment fonctionne le transfert Wi-Fi basé sur un navigateur

Des outils comme [OneTimeDrop](/) utilisent votre connexion Internet (Wi-Fi ou données mobiles) comme support de transfert. Voici le flux :

1. L'ordinateur ouvre une session → génère un code
2. Le téléphone scanne le QR ou saisit le code → rejoint la session
3. Téléchargement de fichiers depuis le téléphone → serveur → disponible en téléchargement sur ordinateur

L'idée clé : **les deux appareils n'ont pas besoin d'être sur le même réseau Wi-Fi**. Chacun a juste besoin de sa propre connexion Internet.

## Guide étape par étape

### Sur n'importe quel ordinateur (aucune installation requise)

1. Ouvrez n'importe quel navigateur (Chrome, Firefox, Safari, Edge)
2. Accédez à [onetimedrop.io](/)
3. Un code QR et un code à 8 chiffres apparaissent immédiatement
4. Gardez cet onglet ouvert

### Sur votre téléphone (aucune application requise)

5. Ouvrez votre appareil photo (iPhone) ou votre scanner QR (Android)
6. Scannez le code QR
7. La page de participation s'ouvre automatiquement dans le navigateur de votre téléphone
8. Appuyez sur **Sélectionner les fichiers** : choisissez ce que vous souhaitez transférer.
9. Télécharger – regardez la barre de progression

### Télécharger (ordinateur)

10. Les fichiers apparaissent dans la liste avec un bouton Télécharger
11. Cliquez pour enregistrer chaque fichier - ils arrivent à la qualité d'origine

## Conseil de confidentialité et de sécurité

> ⚠️ **Avertissement :** OneTimeDrop transmet des fichiers sur Internet (et non en peer-to-peer). Les fichiers sont accessibles uniquement avec votre jeton de session et sont automatiquement supprimés après 10 minutes. Ne transférez pas de documents sensibles sur des ordinateurs publics.

##FAQ

**Q : Cela fonctionne-t-il si les appareils se trouvent sur des réseaux Wi-Fi différents ?**  
R : Oui, ils ont juste besoin d’un accès Internet.

**Q : Est-ce que cela fonctionne sur les données mobiles ?**  
R : Oui. Les données mobiles fonctionnent pour les deux appareils.

**Q : Y a-t-il une limite de vitesse ?**  
R : La vitesse de transfert dépend de votre connexion Internet. Un fichier de 5 Mo est généralement transféré en 1 à 5 secondes.
