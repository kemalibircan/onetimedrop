## Pourquoi le transfert PDF est différent

Les PDF sont souvent importants : contrats, cartes d'embarquement, factures, présentations. Vous ne voulez pas qu'ils soient compressés. Vous ne voulez pas qu'ils soient enregistrés sur des serveurs de messagerie. Et vous les voulez sur le bon ordinateur dès maintenant.

Voici comment procéder correctement.

## La meilleure méthode : transfert de navigateur (le plus rapide, le plus privé)

[OneTimeDrop](/) est spécialement conçu pour ce cas d'utilisation :

1. Ouvrez [onetimedrop.io](/) sur votre ordinateur portable : le code et le code QR apparaissent instantanément
2. Scannez le QR avec votre téléphone (ou visitez [onetimedrop.io/join](/join) et saisissez le code)
3. Sur votre téléphone, appuyez sur **Sélectionner des fichiers** : choisissez votre PDF.
4. Le téléchargement se termine en quelques secondes (PDF typique : 1 à 5 Mo)
5. Cliquez sur Télécharger sur l'ordinateur portable

Le PDF arrive exactement tel qu'il a quitté votre téléphone. Pas de compression, pas de connexion au compte.

## Étape par étape pour les scénarios courants

### Carte d'embarquement PDF
1. Recherchez le PDF dans votre e-mail ou vos téléchargements
2. Suivez les étapes OneTimeDrop ci-dessus
3. Ouvrez le PDF téléchargé dans Adobe Reader ou un navigateur pour confirmer qu'il semble correct

### Contrat de travail ou document juridique
1. Utilisez OneTimeDrop uniquement sur un réseau privé de confiance
2. Après le téléchargement, utilisez immédiatement « Effacer la session » pour supprimer les fichiers du serveur
3. Ne transférez jamais de documents juridiques sur les ordinateurs des bibliothèques publiques

### Diapositives de cours (PPTX → PDF)
1. Convertissez d'abord sur votre téléphone : dans PowerPoint Mobile, utilisez Exporter → PDF
2. Transfert via OneTimeDrop
3. Ouvrez sur l'ordinateur portable

## Conseil de confidentialité et de sécurité

> ⚠️ **Avertissement :** Les fichiers sur OneTimeDrop sont automatiquement supprimés après 10 minutes. Si vous transférez des PDF sensibles (médicaux, juridiques, financiers), assurez-vous que vous êtes sur un réseau sécurisé et utilisez une fenêtre de navigateur privée/incognito sur les ordinateurs partagés. Supprimez le fichier des téléchargements après utilisation.

## Conseil de pro : Vérifiez le PDF après le téléchargement

Ouvrez toujours le PDF téléchargé et faites-le défiler rapidement pour vous assurer que toutes les pages sont correctement transférées avant de fermer la session.

##FAQ

**Q : Quelle est la taille maximale du PDF que je peux transférer ?**  
R : 50 Mo. La plupart des PDF sont bien en dessous de cette limite.

**Q : Le PDF sera-t-il protégé par mot de passe après le transfert ?**  
R : OneTimeDrop transfère les fichiers tels quels. Si votre PDF a un mot de passe, il le conservera.

**Q : Puis-je transférer plusieurs PDF à la fois ?**  
R : Oui : sélectionnez plusieurs fichiers dans le sélecteur de fichiers. Jusqu'à 20 par séance.
