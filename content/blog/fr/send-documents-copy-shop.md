## Le défi du Copy Shop

Vous avez tout sur votre téléphone : le formulaire dont vous avez besoin, le billet que vous avez réservé, le CV que vous avez mis à jour à 23h. Le magasin de photocopie dispose d'un ordinateur, d'une imprimante et d'un employé qui attend que vous envoyiez le fichier par courrier électronique à une adresse que vous aurez probablement mal saisie.

Il existe une bien meilleure façon.

## Méthode : transfert instantané de code QR

Voici comment utiliser [OneTimeDrop](/) dans un magasin de photocopie en moins d'une minute :

### Sur l'ordinateur du photocopieur
1. Ouvrez Chrome ou n'importe quel navigateur
2. Accédez à [onetimedrop.io](/)
3. Un code à 8 chiffres et un code QR apparaissent : ne touchez à rien d'autre

### Sur votre téléphone
4. Ouvrez votre appareil photo et scannez le code QR
5. Appuyez sur le lien qui apparaît – il ouvre automatiquement OneTimeDrop
6. Appuyez sur **Sélectionner les fichiers** et choisissez votre document
7. Attendez 2 à 5 secondes pour le téléchargement

### De retour sur l'ordinateur
8. Votre fichier apparaît dans la liste des fichiers
9. Cliquez sur **Télécharger**
10. Ouvrez-le et imprimez

## Quels documents pouvez-vous transférer ?

- Fichiers PDF (recommandé pour l'impression)
- Documents Word (.docx)
- Fichiers images (.jpg, .png, .heic)
- Présentations PowerPoint

> **Conseil de pro :** Si vous ne l'avez pas déjà fait, convertissez votre document au format PDF avant de le transférer. Les PDF s'impriment exactement comme prévu, quel que soit le logiciel dont dispose l'ordinateur de l'atelier de photocopie.

## Comment convertir en PDF sur votre téléphone

**iPhone :** Ouvrez le document → Partager → Imprimer → Pincer pour agrandir l'aperçu → Appuyez à nouveau sur Partager → Enregistrer au format PDF

**Android (avec Google Chrome) :** Ouvrez le document → Partager → Imprimer → Sélectionnez « Enregistrer au format PDF »

**Google Docs :** Appuyez sur le menu à trois points → Partager et exporter → Envoyer une copie → PDF

## Liste de contrôle de confidentialité pour les ordinateurs publics

- [ ] Utilisez le mode incognito/privé si possible
- [ ] Après l'impression, supprimez le fichier téléchargé du dossier Téléchargements
- [ ] Fermez la session OneTimeDrop (ou attendez l'expiration automatique dans 10 min)
- [ ] Ne laissez pas l'ordinateur enregistrer les mots de passe ou remplir automatiquement les données

> ⚠️ **Avertissement :** Les fichiers OneTimeDrop sont automatiquement supprimés après 10 minutes. Cependant, la copie téléchargée reste sur le disque dur de l'ordinateur jusqu'à ce qu'elle soit supprimée manuellement. Supprimez toujours votre fichier de l'ordinateur après l'impression.

## FAQ pour l'utilisation de Copy Shop

**Q : Que faire si l'ordinateur du photocopieur ne parvient pas à ouvrir mon format de fichier ?**  
R : Convertissez d'abord en PDF sur votre téléphone. Les PDF sont universellement pris en charge.

**Q : Puis-je envoyer plusieurs fichiers pour imprimer un document entier avec des images ?**  
R : Oui, sélectionnez plusieurs fichiers à la fois dans le sélecteur de téléchargement.

**Q : Que faire si l'ordinateur du photocopieur n'a pas Internet ?**  
R : OneTimeDrop nécessite Internet. Dans ce cas, demandez-leur s'ils disposent d'une clé USB que vous pouvez utiliser pour transférer depuis leur ordinateur, ou utilisez le courrier électronique comme solution de secours.

**Q : OneTimeDrop est-il sécurisé sur un ordinateur public ?**  
R : Le transfert est sécurisé (jeton de session protégé), mais le fichier téléchargé restera sur l'ordinateur jusqu'à sa suppression. Suivez la liste de contrôle de confidentialité ci-dessus.
