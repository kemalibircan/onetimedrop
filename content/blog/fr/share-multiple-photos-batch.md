## Pourquoi le transfert de photos par lots est délicat

Envoyer une photo par e-mail est simple. Envoyer 15 photos haute résolution est une autre histoire :
- La taille des e-mails est limitée à 25 Mo : un lot de photos HEIC arrive rapidement
- WhatsApp compresse chaque image (et prend une éternité pour mettre 20 photos en file d'attente)
- Le partage Google Photos nécessite que le destinataire dispose d'un compte
- AirDrop ne fonctionne que sur Apple et nécessite de la proximité

Voici comment procéder correctement.

## Transfert par lots avec OneTimeDrop

[OneTimeDrop](/) prend en charge la sélection et le téléchargement de plusieurs fichiers à la fois, jusqu'à 20 fichiers par session (50 Mo chacun).

### Étape par étape : Envoi de 20 photos

1. Sur l'ordinateur, ouvrez [onetimedrop.io](/) — votre code et votre QR apparaissent
2. Sur votre téléphone, scannez le QR (ou accédez à [onetimedrop.io/join](/join))
3. Appuyez sur **Sélectionner des fichiers**
4. Dans le sélecteur de photos, appuyez longuement sur la première photo, puis appuyez sur chaque photo supplémentaire (sur iOS/Android, la sélection multiple fonctionne avec le sélecteur natif)
5. Téléchargez tous les fichiers – la barre de progression s'affiche par fichier
6. Sur l'ordinateur, chaque fichier possède son propre bouton Télécharger
7. Ou cliquez sur **Télécharger tout (ZIP)** pour les obtenir dans une seule archive.

### Astuce : sélection multiple sur iPhone dans le sélecteur de fichiers

Lorsque OneTimeDrop ouvre le sélecteur de fichiers iPhone, appuyez sur **Parcourir** → **Photos** → appuyez sur **Sélectionner** en haut à droite → appuyez sur chaque photo. Toutes les photos sélectionnées sont téléchargées ensemble.

### Astuce : sélection multiple Android

Dans le sélecteur de fichiers Android d'un navigateur, appuyez longuement sur la première photo, puis appuyez sur les autres pour les sélectionner. Ils seront tous téléchargés en une seule fois.

## Comment s'organiser avant le transfert

Si vous envoyez un lot important, pensez d'abord à créer un dossier sur votre téléphone :

1. Sur iPhone : application Fichiers → iCloud/Sur mon téléphone → Créer un dossier → déplacer des fichiers vers
2. Sur Android : application Fichiers par Google → Créer un dossier → déplacer des photos vers

Cela vous aide à garder une trace de ce que vous avez téléchargé.

## Quand 20 fichiers ne suffisent pas

OneTimeDrop prend en charge 20 fichiers par session. Si vous avez besoin de plus :
1. Transférez les 20 premiers et téléchargez-les
2. Cliquez sur "Effacer la session" sur le bureau
3. Téléchargez le prochain lot sur votre téléphone

Vous pouvez également compresser d'abord les fichiers sur votre téléphone pour réduire le nombre d'éléments (par exemple, 50 photos → 1 fichier zip).

> ⚠️ **Avertissement :** Les fichiers OneTimeDrop sont automatiquement supprimés après 10 minutes. Si vous téléchargez un lot volumineux, démarrez les téléchargements rapidement après le téléchargement.

##FAQ

**Q : Puis-je télécharger un fichier zip de photos ?**  
R : Oui ! Si vous compressez d'abord des photos sur votre téléphone, un seul fichier zip volumineux compte comme un téléchargement.

**Q : Le téléchargement de 20 photos à la fois ralentit-il les choses ?**  
R : Les téléchargements s'effectuent de manière séquentielle dans le navigateur, les uns après les autres. Chaque photo prend 1 à 5 secondes selon sa taille.

**Q : Les données EXIF ​​des photos (emplacement, horodatage) seront-elles transférées intactes ?**  
R : Oui. OneTimeDrop transfère les fichiers tels quels, y compris toutes les métadonnées.
