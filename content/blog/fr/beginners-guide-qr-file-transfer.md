## Qu'est-ce qu'un code QR ?

Un code QR (code de réponse rapide) est le motif de code-barres carré que vous avez probablement repéré sur les menus des restaurants, les emballages de produits et les panneaux d'affichage. Lorsque vous en numérisez un avec l’appareil photo de votre téléphone, un site Web s’ouvre généralement automatiquement.

Lors du transfert de fichiers, les codes QR sont utilisés comme raccourci : au lieu de saisir manuellement une longue adresse Web ou un code, il vous suffit de scanner le carré avec votre appareil photo et vous êtes instantanément connecté.

## Pourquoi les codes QR facilitent le transfert de fichiers

Avant que les codes QR ne soient courants, partager un lien de session signifiait :
1. Taper une URL sur le clavier de votre téléphone
2. Rechercher et saisir un code long
3. Ou brancher un câble USB

Avec un code QR, il vous suffit de pointer votre appareil photo. Une seconde et vous êtes connecté.

## Ce dont vous aurez besoin

- Un ordinateur (ordinateur portable ou de bureau) avec n'importe quel navigateur moderne
- Un smartphone (iPhone ou Android) avec une caméra fonctionnelle
- Accès Internet sur les deux appareils

C'est tout. Pas de câble USB. Aucun téléchargement d'application. Aucun compte.

## Votre premier transfert de fichiers QR Code : étape par étape

### Partie 1 : Configuration sur l'ordinateur

1. Ouvrez votre navigateur (Chrome, Firefox, Edge ou Safari)
2. Tapez dans la barre d'adresse : **onetimedrop.io** et appuyez sur Entrée.
3. Vous verrez la page d'accueil de OneTimeDrop
4. Un grand code QR carré apparaît à gauche et un code numérique à 8 chiffres à droite.
5. Laissez cette page ouverte – ne la fermez pas et ne l'actualisez pas

### Partie 2 : Connectez votre téléphone

**Si l'appareil photo de votre téléphone analyse automatiquement le QR :**
6. Ouvrez votre application appareil photo
7. Pointez l'appareil photo vers le code QR sur l'écran de l'ordinateur
8. Restez immobile pendant une seconde : une notification ou une bannière devrait apparaître en haut de l'écran de votre téléphone.
9. Appuyez sur cette notification – elle ouvre automatiquement une nouvelle page de navigateur sur votre téléphone

**Si vous préférez saisir le code :**
6. Sur votre téléphone, ouvrez un navigateur
7. Accédez à **onetimedrop.io/join**
8. Tapez le code à 8 chiffres affiché sur l'écran de l'ordinateur
9. Appuyez sur « Se connecter au bureau »

### Partie 3 : Téléchargez votre fichier

10. Sur votre téléphone, vous verrez un bouton bleu « Sélectionner les fichiers » : appuyez dessus.
11. Choisissez une photo ou un document sur votre téléphone
12. Le téléchargement démarre automatiquement — vous verrez le nom du fichier et une barre de progression
13. Lorsqu'il est indiqué « Terminé » (ou que vous voyez une coche), le fichier est arrivé sur l'ordinateur.

### Partie 4 : Télécharger sur l'ordinateur

14. Sur l'ordinateur, votre fichier apparaît dans une liste
15. Cliquez sur le bouton orange "Télécharger" à côté de votre fichier
16. Le fichier est enregistré dans votre dossier Téléchargements
17. Ouvrez-le à partir de là – c'est fait !

## Qu'arrive-t-il à mon dossier ?

Au bout de 10 minutes, OneTimeDrop supprime automatiquement tous les fichiers de ses serveurs. Il s'agit d'une fonctionnalité de confidentialité : les fichiers ne traînent pas. Alors téléchargez ce dont vous avez besoin avant l’expiration de la session.

Le compte à rebours est visible sur l'écran de l'ordinateur.

## Erreurs courantes (et comment les corriger)

**"Le code QR n'a pas été scanné"**  
→ Assurez-vous qu'il y a suffisamment de lumière. Tenez votre téléphone plus stable. Rapprochez-vous ou éloignez-vous légèrement.

**"Les 10 minutes se sont écoulées avant le téléchargement"**  
→ Actualisez la page de l'ordinateur et démarrez une nouvelle session. Téléchargez et téléchargez à nouveau.

**"Je ne trouve pas le fichier que j'ai téléchargé"**  
→ Sur la plupart des ordinateurs, les fichiers téléchargés vont dans le dossier Téléchargements. Sous Windows : Explorateur de fichiers → Téléchargements. Sur Mac : Finder → Téléchargements.

## Conseil de confidentialité et de sécurité

> ⚠️ **Avertissement :** Les fichiers OneTimeDrop sont automatiquement supprimés après 10 minutes. Ne téléchargez pas de photos d'identité personnelles, de coordonnées bancaires ou de documents médicaux, en particulier sur des ordinateurs que vous ne possédez pas.

##FAQ

**Q : Que faire si je ne vois pas de scanner QR sur mon téléphone ?**  
R : La plupart des téléphones modernes (iPhone, Android 9+) l’ont intégré à l’application Appareil photo par défaut. Sinon, téléchargez une application gratuite de scanner QR.

**Q : Puis-je l'utiliser pour envoyer des fichiers dans l'autre sens (de l'ordinateur au téléphone) ?**  
R : Pas encore – OneTimeDrop se concentre actuellement sur téléphone → ordinateur. Nous travaillons dans le sens inverse.

**Q : Est-ce que cela fonctionne également sur les tablettes ?**  
R : Oui : les tablettes équipées d'appareils photo et de navigateurs peuvent scanner les codes QR OneTimeDrop tout comme les téléphones.

**Q : Est-ce compliqué pour les débutants ?**  
R : Nous l’avons conçu pour qu’il s’agisse de l’option la plus simple disponible. La plupart des gens effectuent leur premier transfert en moins de 2 minutes.
