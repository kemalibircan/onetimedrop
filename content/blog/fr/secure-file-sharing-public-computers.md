## Les risques des ordinateurs publics

Les ordinateurs publics présentent de sérieux risques pour la vie privée :
- Les **Keyloggers** peuvent enregistrer les mots de passe que vous saisissez
- **L'historique du navigateur** n'est souvent pas effacé entre les utilisateurs
- Les **fichiers téléchargés** peuvent persister dans le dossier Téléchargements
- Les **sessions du navigateur** peuvent ne pas être correctement déconnectées

Malgré ces risques, vous n’avez parfois pas le choix : une bibliothèque, un photocopieur, un centre d’affaires dans un hôtel ou un ordinateur de travail partagé. Voici comment rester en sécurité.

## ✅ La liste de contrôle pour le transfert de fichiers sur ordinateur public

### Avant de commencer
- [ ] Utiliser le mode de navigation incognito/privé (empêche l'historique, les cookies et le remplissage automatique)
- [ ] Vérifiez qui est autour de vous (surf sur l'épaule)
- [ ] Vérifiez que vous êtes sur le vrai site Web (vérifiez l'URL complète dans la barre d'adresse)

### Pendant le transfert
- [ ] Transférez uniquement ce dont vous avez besoin — ne téléchargez pas de fichiers très sensibles
- [ ] Ne vous connectez à aucun compte personnel (e-mail, Google, etc.)
- [ ] Ne cliquez pas sur "Se souvenir de moi" ou "Enregistrer le mot de passe" dans les invites

### Après le téléchargement
- [ ] Téléchargez votre fichier
- [ ] Imprimez immédiatement si tel est l'objectif
- [ ] **Supprimez le fichier téléchargé du dossier Téléchargements**
- [ ] Effacez la session dans OneTimeDrop (ou attendez 10 minutes pour l'expiration automatique)
- [ ] Fermez tous les onglets du navigateur
- [ ] Effacer l'historique du navigateur et les cookies (si possible) : Paramètres → Effacer les données de navigation

### Si vous vous êtes connecté accidentellement
- [ ] Déconnectez-vous de chaque compte
- [ ] Révoquer les sessions de votre téléphone (par exemple, Google : Sécurité → Appareils)

## Utiliser OneTimeDrop en toute sécurité sur des ordinateurs publics

[OneTimeDrop](/) est particulièrement adapté aux ordinateurs publics car :
- Vous ne saisissez aucun mot de passe
- Vous ne vous connectez à aucun compte
- La session est identifiée par un jeton généré aléatoirement (pas vos identifiants)
- Fichiers supprimés automatiquement après 10 minutes

Mais n'oubliez pas : le fichier que vous téléchargez existe toujours sur le disque dur de l'ordinateur.

## Ce qu'il ne faut PAS transférer sur les ordinateurs publics

Même avec une méthode de transfert sécurisée, évitez de télécharger :
- Pièces d'identité gouvernementales (passeport, permis de conduire)
- Documents bancaires
- Dossiers médicaux
- Des photos très personnelles qui pourraient vous gêner

> ⚠️ **Avertissement :** Les fichiers OneTimeDrop sont automatiquement supprimés du serveur après 10 minutes. Cependant, tout fichier que vous téléchargez sur un ordinateur public persiste localement jusqu'à sa suppression. Supprimez toujours manuellement les fichiers téléchargés des ordinateurs partagés.

##FAQ

**Q : Le mode navigation privée est-il suffisant pour sécuriser les ordinateurs publics ?**  
R : Le mode navigation privée empêche l'historique du navigateur et le stockage des cookies, mais ne protège pas contre les enregistreurs de frappe ou autres logiciels malveillants installés sur l'ordinateur.

**Q : Que faire si je dois me connecter à Gmail pour obtenir un fichier ?**  
R : Connectez-vous en mode navigation privée, téléchargez ce dont vous avez besoin, puis déconnectez-vous immédiatement. Après votre session, révoquez cette session de votre appareil principal.

**Q : Le transfert de fichiers lui-même est-il sécurisé ?**  
R : Le transfert vers OneTimeDrop est crypté HTTPS. Le risque ne réside pas dans le transfert, mais dans ce qui se passe une fois le fichier téléchargé.
