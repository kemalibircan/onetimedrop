## Première question : qu'est-ce qui est réellement cassé ?

Un téléphone cassé peut signifier beaucoup de choses. Si l’écran est endommagé mais que le tactile fonctionne toujours, vous disposez encore de plusieurs bonnes options. Si le port est mort mais que le téléphone s'allume et se connecte au Wi-Fi, le transfert sans fil peut vous sauver. Si le téléphone ne démarre pas du tout, vos options se réduisent rapidement.

Commencez donc par séparer les dommages à l’écran des pannes d’alimentation ou de stockage.

## Flux de travail de récupération dans le meilleur des cas

Si le téléphone se déverrouille toujours et se connecte au Wi-Fi :

1. Ouvrez un navigateur ou un point d'accès Fichiers/Photos sur le téléphone.
2. Ouvrez [OneTimeDrop](/) sur un ordinateur à proximité.
3. Scannez le code QR si la caméra fonctionne toujours ou saisissez le code de session manuellement.
4. Transférez d'abord les fichiers les plus importants : pièces d'identité, tickets, contrats, photos récentes, scans.
5. Sauvegardez-les immédiatement après le téléchargement.

## Si l'écran fonctionne à peine

Donnez la priorité uniquement aux fichiers dont vous avez besoin aujourd’hui. Un appareil endommagé peut tomber en panne complètement plus tard, c'est pourquoi un tri urgent compte plus qu'une organisation parfaite.

## Quand le Wi-Fi ne vous aide pas

Le transfert Wi-Fi ne résoudra pas une batterie déchargée, un stockage défaillant ou un téléphone qui ne peut pas se déverrouiller. Dans ces cas, vous aurez peut-être besoin d'un atelier de réparation, d'un remplacement d'écran ou d'un chemin de récupération spécifique au fournisseur.

| Situation | Le Wi-Fi peut-il aider ? | Pourquoi |
|---|---|---|
| Écran fissuré, téléphone toujours utilisable | Oui | Vous pouvez toujours démarrer un transfert |
| Port de chargement endommagé | Souvent oui | Le chemin sans fil évite le port |
| Le téléphone ne s'allume pas | Non | Le stockage est inaccessible |

##FAQ

**Q : Dois-je tout transférer en même temps ?**  
R : Non. Obtenez d’abord les fichiers irremplaçables pendant que le téléphone est encore utilisable.

**Q : Que faire si la caméra est également cassée ?**  
R : Utilisez le code de session manuel au lieu de scanner QR si l'appareil peut toujours ouvrir la page de participation.

**Q : La synchronisation dans le cloud est-elle meilleure ici ?**  
R : Seulement s’il était déjà configuré avant le dommage. Sa configuration sur un appareil défaillant peut être trop lente.
