## Die Risiken öffentlicher Computer

Öffentliche Computer stellen ein ernstes Risiko für die Privatsphäre dar:
- **Keylogger** können von Ihnen eingegebene Passwörter aufzeichnen
- Der **Browserverlauf** wird häufig nicht zwischen Benutzern gelöscht
- **Heruntergeladene Dateien** können im Ordner „Downloads“ verbleiben
- **Browsersitzungen** werden möglicherweise nicht ordnungsgemäß abgemeldet

Trotz dieser Risiken haben Sie manchmal keine andere Wahl – eine Bibliothek, einen Copyshop, ein Hotel-Businesscenter oder einen gemeinsam genutzten Arbeitscomputer. So bleiben Sie auf der sicheren Seite.

## ✅ Die Checkliste für die Dateiübertragung auf öffentlichen Computern

### Bevor Sie beginnen
- [ ] Inkognito-/privaten Browsermodus verwenden (verhindert Verlauf, Cookies und automatisches Ausfüllen)
- [ ] Überprüfen Sie, wer in Ihrer Nähe ist (Schultersurfen)
- [ ] Vergewissern Sie sich, dass Sie sich auf der echten Website befinden (überprüfen Sie die vollständige URL in der Adressleiste)

### Während der Übertragung
- [ ] Übertragen Sie nur das, was Sie benötigen – laden Sie keine besonders sensiblen Dateien hoch
- [ ] Melden Sie sich nicht bei persönlichen Konten an (E-Mail, Google usw.)
- [ ] Klicken Sie bei Eingabeaufforderungen nicht auf „Angemeldet bleiben“ oder „Passwort speichern“.

### Nach dem Herunterladen
- [ ] Laden Sie Ihre Datei herunter
- [ ] Sofort drucken, wenn das das Ziel ist
- [ ] **Löschen Sie die heruntergeladene Datei aus dem Download-Ordner**
- [ ] Löschen Sie die Sitzung in OneTimeDrop (oder warten Sie 10 Minuten auf den automatischen Ablauf)
- [ ] Schließen Sie alle Browser-Registerkarten
- [ ] Browserverlauf und Cookies löschen (falls möglich): Einstellungen → Browserdaten löschen

### Wenn Sie sich versehentlich angemeldet haben
- [ ] Melden Sie sich von jedem Konto ab
- [ ] Sitzungen von Ihrem Telefon widerrufen (z. B. Google: Sicherheit → Geräte)

## OneTimeDrop sicher auf öffentlichen Computern verwenden

[OneTimeDrop](/) eignet sich besonders gut für öffentliche Computer, weil:
- Sie geben keine Passwörter ein
- Sie melden sich bei keinem Konto an
- Die Sitzung wird durch ein zufällig generiertes Token identifiziert (nicht durch Ihre Anmeldeinformationen).
- Dateien werden nach 10 Minuten automatisch gelöscht

Aber denken Sie daran: Die heruntergeladene Datei ist noch auf der Festplatte des Computers vorhanden.

## Was NICHT auf öffentlichen Computern übertragen werden sollte

Vermeiden Sie auch bei einer sicheren Übertragungsmethode das Hochladen von:
- Regierungsausweise (Reisepass, Führerschein)
- Bankdokumente
- Krankenakten
- Sehr persönliche Fotos, die Ihnen peinlich wären

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch vom Server gelöscht. Allerdings bleibt jede Datei, die Sie auf einen öffentlichen Computer herunterladen, lokal bestehen, bis sie gelöscht wird. Entfernen Sie heruntergeladene Dateien immer manuell von gemeinsam genutzten Computern.

## FAQ

**F: Reicht der Inkognitomodus aus, um öffentliche Computer sicher zu machen?**  
A: Der Inkognito-Modus verhindert den Browserverlauf und die Speicherung von Cookies, bietet jedoch keinen Schutz vor Keyloggern oder anderer auf dem Computer installierter Malware.

**F: Was passiert, wenn ich mich bei Gmail anmelden muss, um eine Datei zu erhalten?**  
A: Melden Sie sich im Inkognito-Modus an, laden Sie herunter, was Sie benötigen, und melden Sie sich dann sofort ab. Widerrufen Sie nach Ihrer Sitzung diese Sitzung von Ihrem primären Gerät.

**F: Ist die Dateiübertragung selbst sicher?**  
A: Die Übertragung zu OneTimeDrop erfolgt HTTPS-verschlüsselt. Das Risiko liegt nicht in der Übertragung, sondern darin, was passiert, nachdem die Datei heruntergeladen wurde.
