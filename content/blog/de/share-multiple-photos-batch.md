## Warum die Stapel-Fotoübertragung schwierig ist

Das Versenden eines Fotos per E-Mail ist ganz einfach. Das Versenden von 15 hochauflösenden Fotos ist eine andere Geschichte:
- E-Mails sind auf 25 MB begrenzt – ein Stapel HEIC-Fotos kommt schnell an
- WhatsApp komprimiert jedes Bild (und es dauert ewig, bis 20 Fotos in der Warteschlange stehen)
- Für die Freigabe von Google Fotos muss der Empfänger über ein Konto verfügen
- AirDrop funktioniert nur auf Apple und benötigt Nähe

So machen Sie es richtig.

## Stapelübertragung mit OneTimeDrop

[OneTimeDrop](/) unterstützt das gleichzeitige Auswählen und Hochladen mehrerer Dateien, bis zu 20 Dateien pro Sitzung (jeweils 50 MB).

### Schritt-für-Schritt: 20 Fotos senden

1. Öffnen Sie auf dem Computer [onetimedrop.io](/) – Ihr Code und QR werden angezeigt
2. Scannen Sie auf Ihrem Telefon den QR (oder gehen Sie zu [onetimedrop.io/join](/join)).
3. Tippen Sie auf **Dateien auswählen**
4. Tippen Sie in der Fotoauswahl auf das erste Foto und halten Sie es gedrückt. Tippen Sie dann auf jedes weitere Foto (unter iOS/Android funktioniert die Mehrfachauswahl mit der nativen Auswahl).
5. Laden Sie alle Dateien hoch – der Fortschrittsbalken wird pro Datei angezeigt
6. Auf dem Computer verfügt jede Datei über eine eigene Download-Schaltfläche
7. Oder klicken Sie auf **Alle herunterladen (ZIP)**, um sie in einem Archiv abzurufen

### Tipp: iPhone-Mehrfachauswahl in der Dateiauswahl

Wenn OneTimeDrop die iPhone-Dateiauswahl öffnet, tippen Sie auf **Durchsuchen** → **Fotos** → tippen Sie oben rechts auf **Auswählen** → tippen Sie auf jedes Foto. Alle ausgewählten Fotos werden zusammen hochgeladen.

### Tipp: Android-Mehrfachauswahl

Drücken Sie in der Android-Dateiauswahl eines Browsers lange auf das erste Foto und tippen Sie dann auf weitere, um sie auszuwählen. Sie werden alle auf einmal hochgeladen.

## So organisieren Sie sich vor der Übertragung

Wenn Sie einen großen Stapel senden, sollten Sie zunächst einen Ordner auf Ihrem Telefon erstellen:

1. Auf dem iPhone: Dateien-App → iCloud/Auf meinem Telefon → Ordner erstellen → Dateien verschieben
2. Auf Android: Dateien von Google App → Ordner erstellen → Fotos verschieben

So behalten Sie den Überblick darüber, was Sie hochgeladen haben.

## Wenn 20 Dateien nicht ausreichen

OneTimeDrop unterstützt 20 Dateien pro Sitzung. Wenn Sie mehr benötigen:
1. Übertragen Sie die ersten 20 und laden Sie sie herunter
2. Klicken Sie auf dem Desktop auf „Sitzung löschen“.
3. Laden Sie den nächsten Stapel auf Ihr Telefon hoch

Oder komprimieren Sie die Dateien zuerst auf Ihrem Telefon, um die Anzahl der Elemente zu reduzieren (z. B. 50 Fotos → 1 Zip-Datei).

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch gelöscht. Wenn Sie eine große Menge herunterladen, starten Sie die Downloads sofort nach dem Hochladen.

## FAQ

**F: Kann ich eine ZIP-Datei mit Fotos hochladen?**  
A: Ja! Wenn Sie Fotos zuerst auf Ihrem Telefon komprimieren, zählt eine einzelne große Zip-Datei als ein Upload.

**F: Verlangsamt das gleichzeitige Hochladen von 20 Fotos die Geschwindigkeit?**  
A: Uploads erfolgen nacheinander im Browser – einer nach dem anderen. Jedes Foto dauert je nach Größe 1–5 Sekunden.

**F: Werden die Foto-EXIF-Daten (Standort, Zeitstempel) intakt übertragen?**  
A: Ja. OneTimeDrop überträgt Dateien unverändert, einschließlich aller Metadaten.
