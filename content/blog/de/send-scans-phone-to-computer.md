## Das Scannen von Telefonen ist sehr gut geworden

Moderne Telefone können Dokumente in Sekundenschnelle mit Apps wie den folgenden in PDF scannen:
- **In iOS integriert:** Notizen-App → Kamera → Dokumente scannen
- **Android:** Google Drive App → Scannen
- **Drittanbieter:** Adobe Scan, Microsoft Lens

Das Problem ist nicht das Scannen, sondern die schnelle Übertragung des Scans von Ihrem Telefon auf einen Computer.

## Die schnellste Übertragungsmethode: QR-Sitzung

Nachdem Sie Ihr Dokument gescannt haben, verschieben Sie es mit [OneTimeDrop](/) auf den Computer:

### Schritt 1: Scannen Sie Ihr Dokument

Auf dem iPhone (integriert):
1. Öffnen Sie die Notizen-App
2. Erstellen Sie eine neue Notiz → tippen Sie auf das Kamerasymbol
3. Wählen Sie „Dokumente scannen“
4. Fotografieren Sie Ihr Dokument
5. Tippen Sie auf „Speichern“ – der Scan wird als PDF-Anhang in der Notiz gespeichert
6. Tippen Sie auf PDF → Teilen → In Dateien speichern

Auf Android (Google Drive):
1. Öffnen Sie Google Drive → blaue +-Schaltfläche
2. Wählen Sie „Scannen“
3. Nehmen Sie das Foto auf → schneiden Sie es bei Bedarf zu
4. Speichern – es geht an Google Drive (dann in den Speicher Ihres Telefons herunterladen)

### Schritt 2: Auf den Computer übertragen

1. Öffnen Sie auf dem Computer [onetimedrop.io](/)
2. Scannen Sie auf Ihrem Telefon den QR-Code (oder besuchen Sie [onetimedrop.io/join](/join)).
3. Tippen Sie auf **Dateien auswählen** → finden Sie Ihren Scan unter Dateien/Downloads
4. Hochladen → Klicken Sie auf dem Computer auf Herunterladen

Gesamtzeit vom Scan bis zum Computer: unter 2 Minuten.

## Scanqualität: Welche Einstellungen verwendet werden sollen

| App | Qualität | Ausgabeformat | Notizen |
|---|---|---|---|
| iOS-Notizen | Ausgezeichnet | PDF | Automatische Perspektivkorrektur |
| Google Drive-Scan | Ausgezeichnet | PDF | OCR-indiziert in Drive |
| Adobe Scan | Ausgezeichnet | PDF | Am besten für mehrseitige Dokumente |
| Kamera (Rohfoto) | Variable | JPEG | Keine automatische Bereinigung |

Verwenden Sie zum Drucken oder Archivieren immer die PDF-Ausgabe. Für Fotos von Quittungen/Screenshots ist JPEG in Ordnung.

## Datenschutz- und Sicherheitstipp

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch gelöscht. Wenn Sie vertrauliche Dokumente (Ausweise, Verträge, medizinische Formulare) scannen, stellen Sie sicher, dass Sie sich an einem privaten und vertrauenswürdigen Computer befinden. Löschen Sie die heruntergeladene Datei nach der Verwendung immer von öffentlichen Computern.

## FAQ

**F: Wie exportiere ich einen Scan als PDF unter iOS?**  
A: Scannen Sie Notizen → tippen Sie auf den Scan → tippen Sie auf „Teilen“ → wählen Sie „In Dateien speichern“, um es als PDF zu speichern. Wählen Sie es dann in OneTimeDrop aus.

**F: Kann ich mehrere Seiten in ein PDF scannen?**  
A: Ja. Tippen Sie im iOS Notes-Scanner für jede Seite weiterhin auf den Auslöser. Beim abschließenden Speichern wird ein mehrseitiges PDF erstellt.

**F: Was passiert, wenn mein Scan mehr als 50 MB umfasst?**  
A: Die Scankomprimierung ist äußerst effizient. Ein 10-seitiges Dokument ist normalerweise 1–5 MB groß. Wenn es größer ist, reduzieren Sie die Scanauflösung in Ihren App-Einstellungen.
