## Die Copy-Shop-Challenge

Sie haben alles auf Ihrem Handy: das benötigte Formular, das gebuchte Ticket, den Lebenslauf, den Sie um 23 Uhr aktualisiert haben. Der Kopierladen verfügt über einen Computer, einen Drucker und einen Angestellten, der darauf wartet, dass Sie die Datei per E-Mail an eine Adresse senden, bei der Sie sich wahrscheinlich vertippt haben.

Es gibt einen weitaus besseren Weg.

## Methode: Sofortige QR-Code-Übertragung

So verwenden Sie [OneTimeDrop](/) in weniger als einer Minute in einem Copyshop:

### Auf dem Copyshop-Computer
1. Öffnen Sie Chrome oder einen beliebigen Browser
2. Gehen Sie zu [onetimedrop.io](/)
3. Ein 8-stelliger Code und ein QR-Code werden angezeigt – berühren Sie nichts anderes

### Auf Ihrem Telefon
4. Öffnen Sie Ihre Kamera und scannen Sie den QR-Code
5. Tippen Sie auf den angezeigten Link – OneTimeDrop wird automatisch geöffnet
6. Tippen Sie auf **Dateien auswählen** und wählen Sie Ihr Dokument aus
7. Warten Sie 2–5 Sekunden auf den Upload

### Zurück am Computer
8. Ihre Datei erscheint in der Dateiliste
9. Klicken Sie auf **Herunterladen**
10. Öffnen Sie es und drucken Sie es aus

## Welche Dokumente können Sie übertragen?

- PDF-Dateien (zum Drucken empfohlen)
- Word-Dokumente (.docx)
- Bilddateien (.jpg, .png, .heic)
- PowerPoint-Präsentationen

> **Profi-Tipp:** Konvertieren Sie Ihr Dokument vor der Übertragung in das PDF-Format, falls Sie dies noch nicht getan haben. PDFs werden genau wie vorgesehen gedruckt, unabhängig davon, über welche Software der Copyshop-Computer verfügt.

## So konvertieren Sie auf Ihrem Telefon in PDF

**iPhone:** Öffnen Sie das Dokument → Teilen → Drucken → Ziehen Sie die Finger zusammen, um die Vorschau zu vergrößern → Tippen Sie erneut auf „Teilen“ → Als PDF speichern

**Android (mit Google Chrome):** Öffnen Sie das Dokument → Teilen → Drucken → Wählen Sie „Als PDF speichern“

**Google Docs:** Tippen Sie auf das Dreipunktmenü → Teilen und Exportieren → Kopie senden → PDF

## Datenschutz-Checkliste für öffentliche Computer

- [ ] Verwenden Sie nach Möglichkeit den Inkognito-/Privatmodus
- [ ] Löschen Sie nach dem Drucken die heruntergeladene Datei aus dem Download-Ordner
- [ ] Schließen Sie die OneTimeDrop-Sitzung (oder warten Sie auf den automatischen Ablauf in 10 Minuten)
- [ ] Lassen Sie nicht zu, dass der Computer Passwörter speichert oder Daten automatisch ausfüllt

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch gelöscht. Die heruntergeladene Kopie verbleibt jedoch auf der Festplatte des Computers, bis sie manuell entfernt wird. Löschen Sie Ihre Datei nach dem Drucken immer vom Computer.

## FAQs zur Copyshop-Nutzung

**F: Was passiert, wenn der Copyshop-Computer mein Dateiformat nicht öffnen kann?**  
A: Konvertieren Sie zuerst auf Ihrem Telefon in PDF. PDFs werden allgemein unterstützt.

**F: Kann ich mehrere Dateien senden, um ein ganzes Dokument mit Bildern zu drucken?**  
A: Ja, wählen Sie in der Upload-Auswahl mehrere Dateien gleichzeitig aus.

**F: Was passiert, wenn der Computer des Copyshops kein Internet hat?**  
A: OneTimeDrop erfordert Internet. Fragen Sie in diesem Fall, ob sie über ein USB-Laufwerk verfügen, das Sie für die Übertragung von ihrem Computer verwenden können, oder nutzen Sie E-Mail als Ausweichlösung.

**F: Ist OneTimeDrop auf einem öffentlichen Computer sicher?**  
A: Die Übertragung ist sicher (durch Sitzungstoken geschützt), aber die heruntergeladene Datei bleibt auf dem Computer, bis sie gelöscht wird. Befolgen Sie die Datenschutz-Checkliste oben.
