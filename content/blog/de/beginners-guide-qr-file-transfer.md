## Was ist ein QR-Code?

Ein QR-Code (Quick Response Code) ist das quadratische Barcodemuster, das Sie wahrscheinlich schon auf Speisekarten, Produktverpackungen und Werbetafeln in Restaurants gesehen haben. Wenn Sie eine Website mit der Kamera Ihres Telefons scannen, wird normalerweise automatisch eine Website geöffnet.

Bei der Dateiübertragung werden QR-Codes als Verknüpfung verwendet – anstatt eine lange Webadresse oder einen Code manuell einzugeben, scannen Sie einfach das Quadrat mit Ihrer Kamera und sind sofort verbunden.

## Warum QR-Codes die Dateiübertragung einfacher machen

Bevor QR-Codes üblich waren, bedeutete das Teilen eines Sitzungslinks:
1. Geben Sie eine URL auf der Tastatur Ihres Telefons ein
2. Suchen und Eingeben eines Langcodes
3. Oder ein USB-Kabel anschließen

Mit einem QR-Code richten Sie einfach Ihre Kamera aus. Eine Sekunde und Sie sind verbunden.

## Was Sie brauchen

- Ein Computer (Laptop oder Desktop) mit einem beliebigen modernen Browser
- Ein Smartphone (iPhone oder Android) mit funktionierender Kamera
- Internetzugang auf beiden Geräten

Das ist es. Kein USB-Kabel. Keine App-Downloads. Keine Konten.

## Ihre erste QR-Code-Dateiübertragung: Schritt für Schritt

### Teil 1: Einrichtung am Computer

1. Öffnen Sie Ihren Browser (Chrome, Firefox, Edge oder Safari)
2. Geben Sie in die Adressleiste ein: **onetimedrop.io** und drücken Sie die Eingabetaste
3. Sie sehen die OneTimeDrop-Homepage
4. Links erscheint ein großer quadratischer QR-Code und rechts ein 8-stelliger Zahlencode
5. Lassen Sie diese Seite geöffnet – schließen oder aktualisieren Sie sie nicht

### Teil 2: Verbinden Sie Ihr Telefon

**Wenn Ihre Telefonkamera QR automatisch scannt:**
6. Öffnen Sie Ihre Kamera-App
7. Richten Sie die Kamera auf den QR-Code auf dem Computerbildschirm
8. Halten Sie eine Sekunde lang still – oben auf dem Bildschirm Ihres Telefons sollte eine Benachrichtigung oder ein Banner erscheinen
9. Tippen Sie auf diese Benachrichtigung – sie öffnet automatisch eine neue Browserseite auf Ihrem Telefon

**Wenn Sie lieber den Code eingeben möchten:**
6. Öffnen Sie auf Ihrem Telefon einen Browser
7. Gehen Sie zu **onetimedrop.io/join**
8. Geben Sie den 8-stelligen Code ein, der auf dem Computerbildschirm angezeigt wird
9. Tippen Sie auf „Mit Desktop verbinden“

### Teil 3: Laden Sie Ihre Datei hoch

10. Auf Ihrem Telefon sehen Sie eine blaue Schaltfläche „Dateien auswählen“ – tippen Sie darauf
11. Wählen Sie ein Foto oder Dokument von Ihrem Telefon aus
12. Der Upload startet automatisch – Sie sehen den Dateinamen und einen Fortschrittsbalken
13. Wenn „Fertig“ angezeigt wird (oder ein Häkchen angezeigt wird), ist die Datei auf dem Computer angekommen

### Teil 4: Auf den Computer herunterladen

14. Auf dem Computer erscheint Ihre Datei in einer Liste
15. Klicken Sie neben Ihrer Datei auf die orangefarbene Schaltfläche „Herunterladen“.
16. Die Datei wird in Ihrem Download-Ordner gespeichert
17. Von dort aus öffnen – fertig!

## Was passiert mit meiner Datei?

Nach 10 Minuten löscht OneTimeDrop automatisch alle Dateien von seinen Servern. Dies ist eine Datenschutzfunktion – Dateien bleiben nicht hängen. Laden Sie also herunter, was Sie benötigen, bevor die Sitzung abläuft.

Der Countdown-Timer ist auf dem Computerbildschirm sichtbar.

## Häufige Fehler (und wie man sie behebt)

**„Der QR-Code wurde nicht gescannt“**  
→ Sorgen Sie für ausreichend Licht. Halten Sie Ihr Telefon ruhiger. Bewegen Sie sich etwas näher oder weiter.

**„Die 10 Minuten waren abgelaufen, bevor ich heruntergeladen habe“**  
→ Aktualisieren Sie die Computerseite und starten Sie eine neue Sitzung. Hochladen und erneut herunterladen.

**„Ich kann die hochgeladene Datei nicht finden“**  
→ Auf den meisten Computern werden heruntergeladene Dateien im Ordner „Downloads“ abgelegt. Unter Windows: Datei-Explorer → Downloads. Auf dem Mac: Finder → Downloads.

## Datenschutz- und Sicherheitstipp

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch gelöscht. Laden Sie keine persönlichen Ausweisfotos, Bankdaten oder medizinischen Dokumente hoch – insbesondere nicht auf Computern, die Sie nicht besitzen.

## FAQ

**F: Was passiert, wenn ich auf meinem Telefon keinen QR-Scanner sehe?**  
A: Bei den meisten modernen Telefonen (iPhone, Android 9+) ist es in die Standard-Kamera-App integriert. Wenn nicht, laden Sie eine kostenlose QR-Scanner-App herunter.

**F: Kann ich damit Dateien in die andere Richtung senden (Computer an Telefon)?**  
A: Noch nicht – OneTimeDrop konzentriert sich derzeit auf Telefon → Computer. Wir arbeiten in umgekehrter Richtung.

**F: Funktioniert das auch auf Tablets?**  
A: Ja – Tablets mit Kameras und Browsern können OneTimeDrop-QR-Codes genauso scannen wie Telefone.

**F: Ist das für Anfänger kompliziert?**  
A: Wir haben es so konzipiert, dass es die einfachste verfügbare Option ist. Die meisten Menschen erledigen ihre erste Überweisung in weniger als 2 Minuten.
