## Das Copyshop-Problem

Sie kommen mit einem auf Ihrem Telefon gespeicherten Dokument in eine Druckerei. Der Angestellte zeigt auf einen Computer. Ihr Telefon teilt kein Netzwerk mit ihm. Sie haben kein USB-Laufwerk. Sie möchten sich nicht auf einem öffentlichen Computer bei Ihrem E-Mail-Konto anmelden.

Kommt Ihnen das bekannt vor? Dies ist eine der häufigsten Frustrationen im täglichen digitalen Leben und die Lösung ist einfacher als Sie denken.

## Der einfachste Weg: Browser-Sitzungscode

Tools wie [OneTimeDrop](/) platzieren einen Sitzungscode und einen QR-Code auf dem Bildschirm des Computers, von dem aus Sie drucken möchten. Sie scannen es mit Ihrem Telefon, laden Ihr Dokument hoch und laden es auf den Computer herunter – fertig zum Drucken.

Keine Konten. Keine USB-Kabel. Keine E-Mails an sich selbst.

## Schritt für Schritt in einer Druckerei oder Bibliothek

### Am Computer (Copyshop)

1. Öffnen Sie einen Browser und gehen Sie zu [onetimedrop.io](/)
2. Auf dem Bildschirm werden ein großer QR-Code und ein 8-stelliger Code angezeigt
3. Schließen Sie diese Registerkarte nicht – lassen Sie sie sichtbar

### Auf Ihrem Telefon

4. Öffnen Sie Ihre Kamera-App (iPhone/modernes Android) und scannen Sie den QR-Code
5. Oder gehen Sie zu [onetimedrop.io/join](/join) und geben Sie die 8-stellige Nummer ein
6. Tippen Sie auf **Dateien auswählen** und wählen Sie Ihr Dokument (PDF, Word usw.)
7. Warten Sie, bis der Upload abgeschlossen ist (Sekunden).

### Zurück am Computer

8. Die Datei erscheint in der Dateiliste
9. Klicken Sie auf **Herunterladen** – die Datei wird auf dem Desktop gespeichert
10. Öffnen Sie es und drucken Sie normal

## Welche Dateitypen kann ich zum Drucken senden?

- ✅ PDF – beste Wahl zum Drucken
- ✅ DOCX (Word)
- ✅ PPTX (PowerPoint)
- ✅ Bilder: JPEG, PNG, HEIC

Bevorzugen Sie zum Drucken immer **PDF** – die Formatierung bleibt erhalten, unabhängig davon, welche Software sie öffnet.

## Datenschutz an öffentlichen Computern

> ⚠️ **Haftungsausschluss:** OneTimeDrop-Dateien werden nach 10 Minuten automatisch gelöscht. Seien Sie jedoch auf öffentlichen Computern vorsichtig. Von Ihnen heruntergeladene Dateien bleiben möglicherweise im Download-Ordner, bis sie manuell gelöscht werden. Löschen Sie heruntergeladene Dateien nach dem Drucken immer von gemeinsam genutzten Computern.

### Sicherheitscheckliste für öffentliche Computer
- Verwenden Sie nach Möglichkeit den Privat-/Inkognito-Modus
- Löschen Sie Ihre heruntergeladene Datei nach dem Drucken aus den Downloads
- Geben Sie keine Passwörter oder persönlichen Daten ein
- Schließen Sie alle Browser-Registerkarten, wenn Sie fertig sind

## Häufig gestellte Fragen

**F: Was passiert, wenn der Copyshop-Computer keine PDFs öffnen kann?**  
A: Die meisten modernen Computer verfügen über eine integrierte PDF-Unterstützung. Alternativ können Sie direkt über den integrierten Viewer von Chrome drucken.

**F: Was passiert, wenn der Computer kein Internet hat?**  
A: OneTimeDrop benötigt Internet auf beiden Geräten. Wenn der Computer nicht über eine Internetverbindung verfügt, fragen Sie im Geschäft nach, ob eine USB-Option verfügbar ist.

**F: Wie konvertiere ich meine Datei auf meinem Telefon in PDF?**  
A: Auf dem iPhone: Verwenden Sie die Verknüpfung „Als PDF drucken“ in einer beliebigen App. Auf Android: Mit den meisten Office-Apps können Sie direkt als PDF exportieren.

**F: Ist der 8-stellige Code sicher?**  
A: Der Code wird zufällig generiert und läuft in 10 Minuten ab. Nur die Person mit dem Code kann Ihrer Sitzung beitreten.
