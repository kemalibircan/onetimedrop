## USB Drives: Convenient Until You Forget One

USB drives are the original portable file transfer tool. Fast, offline, no accounts required — when you have one. But what do you do when you don't?

Here are your best options.

## Option 1: QR Browser Transfer (Best for Phone → Computer)

If the file is on your phone, [OneTimeDrop](/) is the fastest wireless alternative to USB:

1. Open [onetimedrop.io](/) on the computer
2. Scan the QR code on your phone
3. Upload — download on the computer

Zero hardware required. Works on any OS. Takes 30 seconds.

## Option 2: Cloud Storage + Quick Share Link

If the file is already on a cloud service:
- **Google Drive:** Right-click file → Get link → Share
- **Dropbox:** Click file → Share → Create link
- **iCloud:** Right-click on Mac → Share → Copy iCloud link

Send or read the link on the other device. If the file isn't on cloud yet, this requires an upload first.

**Best for:** Large files, sharing over distance, files already in the cloud

## Option 3: Email to Yourself

Old faithful. Works for files under 25MB:
1. On your phone, compose an email to yourself with the file attached
2. Open Gmail/Outlook on the computer, download the attachment

**Best for:** Small documents when other methods aren't available

## Option 4: Bluetooth (Local, No Internet)

If both devices support it and are within 10m:
1. Enable Bluetooth on both
2. Pair them (one-time setup)
3. Use "Send file via Bluetooth" on the sender

**Best for:** Small files between already-paired devices in the same room  
**Not recommended for:** Cross-platform (iOS to Windows), large files, or first-time pairing

## Option 5: AirDrop (Apple Only)

If you're using Apple devices in the same room:
1. Sender: Share → AirDrop
2. Recipient: Accept

Ultra-fast, no internet needed.  
**Major limitation:** Apple devices only.

## FAQ

**Q: Which method works completely offline?**  
A: USB (if you had it), Bluetooth, and AirDrop don't require internet. Everything else does.

**Q: What if I don't have a USB and there's no internet in the room?**  
A: Bluetooth is your best offline cross-device option. Pair the devices and transfer.

**Q: What's the fastest method overall?**  
A: USB or AirDrop (when available). OneTimeDrop is fastest when those aren't options.
